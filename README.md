## The qwerty app
Repository for releases of the qwerty app. Here you can check the latest releases for the qwerty app as well as the history of all previous releases. Check the main [page](https://theqwertyapp.xyz) for more information.

### What is qwerty?
qwerty is a simple screenwritting desktop app whose only goal is to get that great story living inside your headinto the page. So create it, imagine it, write it, rewrite it and then share it with the world.

### No breaks.
With the no breaks function you just type whatever comes out of your mind and qwerty will take care of the rest.

### Mark it.
Create markers to more easily navigate through your script in no time.
